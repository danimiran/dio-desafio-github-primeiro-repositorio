HTML - Forma toda a estrutura das páginas, ele é composto de suas tags, atributos e probliedades

sua estrura:

<!DOCTYPE html>
<html>
	<head>
		<title></title>
	</head>

	<body>

	</body>
</html>

-----------------------------------------------------
<!DOCTYPE html> - indica que estamos utilizando o html na ultima versão para o navegador

<html> - inicio do codigo html
<head> - Cabeçario
<title> - titulo da página
<body> - inicio do corpo, conteúdo
